# Kerbal-Space-Agency
A KSP playlist of music from Kitten Space Agency (as of 11-15-2025) using Soundtrack Editor Forked. The current tracks include: Stardance, Sun Synchronous, Drifting, LEO Sunrise, Liminal Space, and Small Blue Dot.

Dependencies are Soundtrack Editor Forked by linuxgurugamer and its dependencies. Install Soundtrack Editor Forked (and its dependencies normally, then install Kerbal Space Agency into the same KSP root folder, replacing the playlists.cfg when prompted.
